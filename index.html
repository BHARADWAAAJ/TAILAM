Help

Guide to using TAILAM™

Main Tank

Enter gas-in-oil concentrations (ppm) from your main tank lab report and run the analysis. TAILAM applies Duval Triangle 1, Rogers Ratio, IEC 60599, IEEE C57.104, Key Gas/TDCG, Doernenburg and CIGRE methods together and reports a consensus health assessment.

OLTC

Enter gas-in-oil concentrations from the OLTC oil compartment. TAILAM applies Duval Triangle 2, TGC comparison against CIGRE TB 443 limits, diagnostic ratios and a cross-contamination check against the main tank. Main Tank and OLTC are independent analyses with separate exports.

Engineering Workspace

The left column is the input form; the right column is the engineering workspace. After you run an assessment it answers, in order: is the transformer safe, what is the most probable fault, what should you do next, how healthy is the transformer, why TAILAM concluded this, and what evidence supports it. Expand any "Why?" control for the supporting detail behind a result.

PDF Export

After running an analysis, use "Export Professional PDF" to open a print-ready report in a new window, then use your browser's print dialog to save it as a PDF.

Excel Export

Use "Export Excel" for a formatted workbook with the Duval triangle embedded as an image. If the Excel library cannot load, a CSV file is downloaded instead.
