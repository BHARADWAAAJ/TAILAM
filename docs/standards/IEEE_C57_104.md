# Standard Implementation Note — IEEE C57.104

**Standard:** IEEE C57.104, *IEEE Guide for the Interpretation of Gases
Generated in Mineral Oil-Immersed Transformers.*
**Status in TAILAM:** Implemented. Frozen for Version 1.0.
**This document does not reproduce any text, table, or figure from the
standard.** It describes only how TAILAM's software applies it. Engineers
verifying TAILAM's output should refer to the official IEEE C57.104
document, obtainable from IEEE, for the authoritative thresholds.

## Purpose

IEEE C57.104 provides individual-gas concentration limits and a combined
Total Dissolved Combustible Gas (TDCG) condition classification, used
alongside ratio-based methods to give a severity read on a transformer's
gassing condition.

## Scope of TAILAM's implementation

1. **Individual gas condition limits** —
   `src/js/engine/ieee.js#calcIEEE`. Each of the seven measured gases (H₂,
   CH₄, C₂H₆, C₂H₄, C₂H₂, CO, CO₂) is compared against its own three
   condition boundaries; the worst individual-gas condition sets the
   overall Condition 1–4 classification.
2. **TDCG condition bands** —
   `src/js/engine/keygas.js#calcKeyGas` (the `TDCG`/`tdcgCond` fields).
   TDCG is computed as the sum of the six combustible gases (H₂, CH₄,
   C₂H₆, C₂H₄, C₂H₂, CO) and classified into one of four condition bands.
3. **Key Gas method** — also in `keygas.js`, TAILAM implements the
   dominant-gas pattern recognition (which combustible gas or gas group is
   generated in the greatest proportion) that IEEE C57.104 describes as a
   qualitative cross-check alongside TDCG.

## How TAILAM implements it

`calcIEEE` evaluates every gas independently against its own limit triple
and returns both the per-gas detail (used in the Raw Calculations table)
and the single worst-case condition number, which is what drives the
Engineering Workspace's IEEE severity display. `calcKeyGas` is a separate,
independent function — TDCG and the dominant-gas pattern are computed from
the same raw gas set but do not depend on the individual-gas condition
result, so the two methods can be read side by side without one silently
overriding the other.

Both functions return plain result objects with no DOM access; the
Diagnostic Methods table and Raw Calculations panels format and display
these results without altering them.

## Limitations

- TAILAM implements the gas-limit and TDCG guidance as encoded in its
  engine; it does not implement the standard's guidance on sampling
  frequency, oil-preservation-system-specific adjustments, or furan
  analysis, which are outside the scope of a DGA calculator.
- Condition classification reflects a single sample; the standard's
  guidance on *rate of gas generation* (trend-based) is not implemented in
  Version 1.0 because TAILAM does not retain historical samples. See
  `docs/release/ROADMAP.md`.
