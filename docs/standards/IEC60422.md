# Standard Implementation Note — IEC 60422

**Standard:** IEC 60422, *Mineral insulating oils in electrical equipment —
Supervision and maintenance guidance.*
**Status in TAILAM:** Partially implemented (dissolved-oxygen
interpretation). Frozen for Version 1.0.
**This document does not reproduce any text, table, or figure from the
standard.** Engineers verifying TAILAM's output should refer to the
official IEC 60422 document for the authoritative guidance.

## Purpose

IEC 60422 provides broader mineral-oil-quality supervision guidance beyond
dissolved combustible gases, including moisture, acidity, and dissolved
oxygen. TAILAM implements the dissolved-oxygen portion of this guidance as
a supplementary oil-condition indicator alongside its combustible-gas
diagnostics.

## Scope of TAILAM's implementation

**Dissolved-oxygen interpretation** —
`src/js/engine/iec.js#interpretO2`. Given a dissolved-O₂ concentration in
ppm (optional input; not required to run an analysis), TAILAM classifies
it into one of four bands (very low / low-normal for a sealed tank /
normal-elevated / high, indicating possible air ingress). This appears in
the Diagnostic Methods table and Raw Calculations as "O₂ Interpretation"
when a value is entered, and contributes an action-plan item when the
result is outside the normal-for-sealed-tank band.

The function is implemented in `iec.js` alongside the IEC 60599 three-ratio
method for code co-location (both are oil-chemistry interpretations reading
the same gas-input form) — it is attributed to IEC 60422, not IEC 60599, in
the UI's reference labels and in this documentation set.

## How TAILAM implements it

`interpretO2` is a pure function returning `null` when no O₂ value has
been entered (the field is optional) and a `{label, desc, cls}` result
object otherwise. It is never a required input and never blocks an
analysis from running.

## Limitations

- TAILAM implements only the dissolved-oxygen supervision guidance from
  this standard. It does not implement moisture-in-oil interpretation,
  acidity/neutralization value guidance, dielectric breakdown voltage
  guidance, or any other IEC 60422 parameter — those are outside the scope
  of a DGA-focused calculator in Version 1.0.
- Oxygen interpretation bands assume a sealed-tank (nitrogen-blanketed or
  hermetically sealed) mineral-oil transformer, as noted in the result
  text; TAILAM does not ask the user to confirm tank type before applying
  them.
